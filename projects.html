  • Quadcopter

    Over the summer of 2015, I researched and built a quadcopter with two other friends. The quadcopter is built from scratch, and is powered by python code run on a raspberry pi as the flight controller. Surrently, we have full control of the four propellers and can read gyrometer inputs. However we have run into complications with actually running code to make the quadcopter fly. The PID algorithm to fly the quadcopter is incredibly complicated and will take more research before actually flying.
