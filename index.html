UCI MedAppJam Submission

Fall 2017

Project Repository
Handoff Helper close

Created an android application to streamline the process of doctors handing off patients to the next doctor on shift. Written in Android Studio using Google Firebase as the backend.

Functionality:

  • Register a personal account linked with an email
  • Log in to the application
  • Add Patient
    • Bed Number
    • Sex
    • Severity
    • Condition
    • Diagnosis
    • Tests Ordered
    • Plan of Care
    • Contingency
  • Add Notes
  • Hand a patient off to another doctor

Created by a team of five in three days.
